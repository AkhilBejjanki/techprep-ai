# techPrep AI

An AI-powered web app built using **Flask** and **Googleâ€™s Gemini API** that helps users prepare for **technical interviews** by generating short, point-wise answers to any question they ask.

---

##  Features

-  **AI-Powered Answers:** Uses Google's Gemini 2.5 model to generate relevant and concise answers.  
-  **Point-Wise Format:** Gives short, readable bullet points instead of long paragraphs.  
-  **Clean Text Output:** Removes markdown symbols for a professional display.  
-  **Secure API Key Handling:** Uses `.env` file to store keys safely.  
-  **Fast & Lightweight:** Built with Flask and Gemini Flash model for instant responses.
